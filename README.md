Course APP

A Node.js + Express based backend application that provides APIs for user and admin functionalities — including authentication, courses, and purchase handling. Built with MongoDB, JWT authentication, and Express routing for modularity and scalability.

⸻

 Features

 User
	•	Signup & Login – Secure authentication using JWT.
	•	View All Courses – Browse all available courses.
	•	Purchase a Course – Enroll in a course and record the purchase.
	•	View Purchased Courses – Access all purchased courses.

 Admin
	•	Signup & Login – Admin authentication with JWT.
	•	Create Course – Add a new course with details and content.

⸻

 Tech Stack
	•	Node.js – Runtime environment
	•	Express.js – Web framework
	•	MongoDB + Mongoose – Database and ODM
	•	jsonwebtoken (JWT) – Authentication
	•	dotenv – Environment variable management
